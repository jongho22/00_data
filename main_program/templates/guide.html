DH_service 사용 가이드입니다.

DH_SERVICE는 뉴스 기사 댓글을 분석하여 사용자가 검색한 키워드의 동향이 긍정적인지, 부정적인지 분석하는 시스템입니다. 하단 설명을 읽고 사용해주시기 바랍니다. 감사합니다.

1. Home(검색) 페이지

(1) 검색할 키워드를 입력하십시오.
(2) 분석하고 싶은 기간을 선택하십시오.
(3) search(검색) 버튼을 클릭해주십시오.


2. 분석 페이지

사용자가 입력한 검색어가 포함된 기사의 댓글을 분석하고 있습니다.


3. 분석 결과 페이지

(1) 사용자가 입력한 검색어를 표시합니다.
(2) 해당 키워드의 긍정적인 척도를 나타내는 그래프입니다.
(3) 해당 키워드의 부정적인 척도를 나타내는 그래프입니다.
(4) 해당 키워드의 관심 척도를 나타내는 그래프입니다.

* 댓글이 없거나, 긍정 또는 부정의 척도가 0인 경우, 해당 날짜의 결과 값이 0으로 표시될 수 있습니다.


4. Result(결과) 페이지

(1) 지난 결과 값을 열람하려면 상단 바에서 Result(결과) 버튼을 누르십시오.
(2) 사용자가 최근에 분석한 결과부터 보여줍니다.
